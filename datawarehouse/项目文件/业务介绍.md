### 业务类型
编码字典定义 (Code Dictionary)
#### 业务线 (Business Line) - 3位 （商品）
Code	Description
101	核心IP产品  # 基于文化IP开发的高价值纪念品和收藏品
102	场景化旅游商品  # 结合旅游场景开发的实用型和体验型商品
103	引流品/虚拟服务  # 低价促销商品和数字化服务产品
#### 渠道 (Channel) - 3位（订单表）
Code	Description
010	抖音  # 短视频和直播电商平台，30%左右
020	微信  # 基于微信生态的电商小程序+微信内的短视频内容和电商平台  20%左右
030	淘宝  # 综合性电商平台 20%左右
031	天猫  # 品牌电商平台  10%左右
040	线下合作  # 景区、旅行社等线下合作伙伴渠道 5%-10%
099	其他  # 其他非主流销售渠道 5%左右
#### 合作模式 (Co-op Model) - 3位 (店铺)
Code	Description
001	自营直销 (Self-operated)  # 品牌方直接运营销售
002	达人分销 (Influencer Distribution)  # 通过KOL/KOC推广销售
003	B2B供货 (B2B Supply)  # 向企业客户批量供货
004	联名合作 (Co-branding)  # 与其他品牌联合开发产品
#### 流量来源类型 (Traffic Source Type) - 3位 (订单表)
Code	Description
001	自然流量  # 未付费的搜索、推荐等流量
002	付费流量  # 通过广告投放获取的流量
003	内容流量  # 通过短视频、图文等内容吸引的流量
004	社交裂变  # 通过用户分享获取的流量
005	线下引流  # 从线下场景引导至线上的流量
099	未知/其他  # 来源不明确的流量
#### 订单类型 (Order Type) - 3位 (订单表)
Code	Description
001	普通销售订单  # 个人消费者的标准订单
002	B端批发订单  # 企业客户的批量采购订单
003	样品/赠品单  # 用于展示或赠送的订单
004	员工内购单  # 企业员工的内部购买订单
005	换货单  # 因产品问题产生的换货订单
### 应用实例与数据落地
场景1: 一位用户通过刷抖音短视频，被种草了一款"场景化旅游商品"（长安伴手礼），点击视频下方的链接，在抖音商城里下单购买。

业务拆解:
业务线: 场景化旅游商品 (102)
渠道: 抖音 (010)
合作模式: 自营直销 (001)
流量来源: 内容流量 (003)
订单类型: 普通销售订单 (001)

场景2: 合作景区的前台，通过内部系统下了一笔批发订单，采购一批"核心IP产品"（兵马俑手办）用于店内销售。

业务拆解:
业务线: 核心IP产品 (101)
渠道: 线下合作 (040)
合作模式: B2B供货 (003)
流量来源: N/A (可记为099)
订单类型: B端批发订单 (002)
